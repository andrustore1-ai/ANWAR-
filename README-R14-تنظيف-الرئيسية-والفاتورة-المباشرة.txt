R14 - تنظيف الرئيسية والفاتورة المباشرة

- حذف بطاقة تقرير الفواتير المباشرة من الصفحة الرئيسية/لوحة المتابعة.
- زر فاوترة مباشر يفتح صفحة كاملة داخل صفحة الفواتير بدل نافذة منبثقة.
- تظهر الفاتورة المباشرة بصنف واحد أولاً، ويوجد زر + إضافة صنف لإضافة أسطر جديدة.
- حذف عبارات التجميل والبيع جملة ومفرق من نموذج الفاتورة المباشرة.
- إصلاح زر المزامنة: عند توفر الإنترنت تتم محاولة المزامنة وتنظيف عداد العمليات الوهمي، والرقم لا يبقى ظاهرًا بدون سبب.
- تحديث كاش Service Worker إلى R14.
