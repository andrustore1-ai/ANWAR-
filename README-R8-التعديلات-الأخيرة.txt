تعديلات R8:
- حذف جملة النظام القديم من صفحة إضافة الصنف.
- حذف حقل باركود إضافي والاكتفاء بـ SKU / باركود مع الكاميرا.
- ترتيب الحقول كل حقلين مقابل بعض في إضافة الصنف، إضافة مشتريات، والنماذج العامة.
- ترتيب صفحة إضافة مشتريات بصريًا بدون تغيير منطقها.
- تثبيت رقم عمليات المزامنة حتى لا يدور مع زر المزامنة.
- حفظ العملاء والموظفين والنماذج العامة محليًا عند عدم توفر الإنترنت.
- عند عودة الإنترنت تتم المزامنة تلقائيًا، ويمكن تشغيلها يدويًا من زر المزامنة.
- تغيير رسالة حفظ الصنف إلى: تم حفظ الصنف.
