تحديث الكاشير والأوفلاين والسجلات

- استبدال ملف الكاشير الأساسي بملف الكاشير النهائي مع ربط الجوال والباركود، مع إبقاء الاسم: الكاشير.html.
- إضافة ملف كاميرا الجوال: كاميرا-الكاشير.html وربطه من زر قارئ الجوال داخل الكاشير.
- تحديث خدمة Service Worker حتى تفتح الصفحات المخزنة أوفلاين بعد أول تحميل، وتعمل الروابط الداخلية بدون طلب تحميل جديد من الإنترنت.
- جعل اسم العميل الجديد في الفاتورة يُحفظ تلقائياً كعميل جديد حتى لو لم يظهر ضمن الاقتراحات.
- إبقاء اسم "زبون نقدي" للكاش فقط، و"زبون تطبيق" لتطبيق فوري عند عدم إدخال اسم.
- إجبار تطبيق لاحق / دين على اسم العميل ورقم الجوال، وترحيل الدين مع بيانات العميل.
- تطوير صفحة الديون: اسم الزبون، رقم الجوال، زر سجل العميل، وزر تم الدفع مع اختيار الحساب.
- تطوير سجل العميل من صفحة العملاء والديون: يعرض الفواتير المكتملة وغير المكتملة، الديون، دفعات السداد.
- إضافة أزرار تصدير سجل العميل: CSV، Excel، وطباعة / PDF.
- تثبيت اختيارات القوائم المنسدلة بعد تحديث الصفحة حتى يغيّر المستخدم الخيار يدوياً.
- توحيد لون الموقع والهيدر والأيقونات والبطاقات على اللون الجديد #409898.
- استبدال رموز الشريط السفلي القديمة بأيقونات SVG حديثة ثابتة حتى لا تظهر رموز/ملصقات مؤقتة أثناء تحميل الصفحة.
